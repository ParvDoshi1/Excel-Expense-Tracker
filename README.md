# Excel Expense Tracker 📊

A simple, single-file Excel expense tracker designed to monitor daily spending, categorize expenses, and maintain personal budgets.

## 📝 Overview
This repository contains a lightweight Excel template built to make personal finance tracking straightforward. Instead of relying on complex software, this tracker uses standard Excel formulas to provide a clear picture of cash flow. This is a very basic Expense Tracker that needs all things to be manually typed.

**Note:** This file is a blank template containing dummy data for demonstration purposes.

## ✨ Features
* **Daily Logging:** Easy-to-use rows for inputting daily expenses (e.g., ₹150 for coffee, ₹500 for stationary).
* **Categorization:** Sort spending into distinct buckets (Food, Transportation, Academics, Entertainment).
* **Automated Totals:** Built-in formulas automatically calculate total spending and Average spending.

## 🚀 How to Use
1. **Download the File:** Click on the `Expense Tracker.xlsx` file above and hit the "Download" button.
2. **Open the File:** Open the downloaded file using Microsoft Excel (or any compatible spreadsheet software like Apple Numbers or Google Sheets).
3. **Clear the Dummy Data:** Delete the example rows provided in the template.
4. **Start Tracking:** Begin entering your own daily expenses. The totals and formulas will update automatically!
